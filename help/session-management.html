üé¨ Ending a Session (GM)

As a Game Master, you don't need to perform any special "save" action. RIFT automatically preserves all session data. However, following these best practices ensures a smooth continuation.

Before Ending the Session

1
Confirm All Players Have Saved

Ask each player to verify their character sheet shows recent changes. The auto-save indicator should show "Saved" status.

2
Note Current Game State

Make a quick note of: current location, active effects, pending initiative, and any temporary conditions on characters.

3
Export JSON Backups (Recommended)

Have each player download their character as JSON. This serves as an offline backup independent of browser storage.

4
Screenshot Key Information

For complex encounters, take a screenshot of the initiative order, battlefield positions, or resource usage.

üí° Pro Tip

The Atmosphere Panel settings (sound, lighting, effects) persist automatically. You can resume with the exact same ambiance.

‚è±Ô∏è Time Factor & Data Persistence

A common concern: "Does my data expire if I don't play for weeks or months?"

Short Answer: No Time Limit

RIFT data does not expire based on time. Your character sheets and settings remain intact indefinitely, as long as:

  • You use the same browser on the same device
  • You haven't cleared browser data (cookies, site data, etc.)
  • The browser hasn't performed an automatic cleanup due to low storage

‚úì Session End Checklist

Use this checklist before ending each session:

For the GM

  • Announce "Session ending in 5 minutes" to allow players to finish actions
  • Note current in-game time, location, and situation
  • Save any atmosphere settings you want to preserve
  • Remind players to download their JSON backups
  • Confirm all players see "Saved" status on their sheets
  • Schedule next session if possible

For Players

  • Verify character sheet shows current HP, spell slots, resources
  • Download JSON backup (Manage Sheet ‚Üí Download JSON)
  • Store backup in cloud folder or email to yourself
  • Note any temporary effects or conditions to remember
